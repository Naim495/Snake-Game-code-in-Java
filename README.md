This is a classic Snake Game implemented in Java using Swing for GUI. The game features:

Snake Movement – Controlled using arrow keys.

Food Eating – The snake grows when it eats food, and the score increases.

Collision Detection – Game ends if the snake hits the walls or itself.

Score Display – Current score is always visible.

Game Over Screen – Shows “Game Over” and final score.

Restart Button – Lets the player restart the game without closing the window.

Technologies Used:

Java SE

Swing (JFrame, JPanel, JButton)

Timer for game loop

Event handling for keyboard input
